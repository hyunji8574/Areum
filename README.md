# Areum

반려견, 반려묘 라이프스타일 브랜드를 주제로 한
개인 작업 사이트입니다.

### 디렉토리 구조
- css
  - scss
    - common
      - _common.scss
      - _footer.scss
      - _header.scss
      - _reset.scss
    - style
      - _brand-layout.scss
      - _brand-layout-responsive.scss
      - _index-layout.scss
      - _index-layout-responsive.scss
    - brand.scss
    - index.scss
  - brand.css
  - index.css
- fonts
- img
- js
- brand.html
- index.html

### 추가 사항

- brand.scss, index.scss는 통합용 파일로, 다른 scss파일을 모두 묶어 하나의 css파일로 내보내는 역할을 합니다.
- -responsive가 붙은 파일은 반응형 을 위한 파일입니다.
